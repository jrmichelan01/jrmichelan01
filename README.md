# Sistema Prisional

Este é um sistema de simulação para controle de portões em um ambiente prisional, desenvolvido em Python.

## Funcionalidades

- **Controle de Portões**: Gerencia o estado dos portões A e B (aberto/fechado).
- **Autenticação**: Verifica placa e RE para liberar acesso.
- **Alarme**: Sistema de alarme para segurança.
- **Timeout**: Limite de tempo para entrada de veículos.

## Como Usar

1. Execute o script Python:
   ```bash
   python app.py
   ```

2. Siga as instruções no terminal para autenticar e controlar os portões.

## Estrutura do Código

- `SistemaPrisional`: Classe principal com métodos para controle dos portões e autenticação.

## Requisitos

- Python 3.x

## Exemplo de Uso

Digite a placa e RE quando solicitado. O sistema abrirá o portão A se autenticado, aguardará o veículo e abrirá o portão B.
